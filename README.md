# Incident & Alerts System

A production-grade incident management system built with Node.js, Express, PostgreSQL, Redis, and Next.js.

## Quick Start

```bash
docker compose up --build
```

The application will be available at:
- Backend API: http://localhost:3000
- Frontend: http://localhost:3001

## Prerequisites

- Docker
- Docker Compose

## Environment Variables

Copy `.env.example` files in both backend and frontend directories and adjust as needed.

## Development

See backend and frontend directories for development setup.
